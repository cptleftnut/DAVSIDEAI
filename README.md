# DAVSIDE AI

En let, mobil-first AI boligagent med Grok Imagine.

**Features:**
- Avanceret boligsøgning med danske boligdata
- Virtual Staging (billeder + video)
- Forhandlingsagent
- Kontrakt-tjek
- Mobil-optimeret UI
- Setup Wizard med API-nøgle validering

**Kør projektet:**
```bash
cargo tauri android dev
```
